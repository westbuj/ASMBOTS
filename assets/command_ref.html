sBotv2 Commands

These commands apply to sBotV2 chipset.
RSET { "Target register", "Value"}
set register TARGET_REG to VALUE.
Example:
RSET TRGX BALX
Set the TRGX register to the value in the BALX register.
DSET{ "source reg 1", "target value 1", "source reg 2","target value 2"}
sets 2 registers at once.
Example:
RSET TRGX BALX TRGY BALY
Set the TRGX and TRGY registers to the value in the BALX and BALY register.
KICK{ "Direction", "Power"}
Kick the ball.
Example:
RSET GOLR 5
Kicks the ball at the angle in param1 at the power in param2.
JULT{ "Number 1", "Number 2","Label"}
Jump if less than
Example:
JULT AUX1 AUX2 :LABEL
Jumps to the specified label if param1 < param2.
JUGT{ "Number 1", "Number 2","Label"}
Jump if greater than.
Example:
JUGT AUX1 AUX2 :LABEL
Jumps to the specified label if param1 > param2";.
JUBT{ "number", "lower","upper",":label"}
Jump if between.
Example:
JUBT AUX1 AUX2 :LABEL
Jumps to the specified label if param3 < param1 < param2.
JUXT{ "number", "lower","upper",":label"}
jump if not between
Example:
JUXT AUX1 AUX2 :LABEL
Jumps to the specified label if number is outside lower and upper.
INC{ "Target Register", "Increment amount"}
Increment
Example:
INC RET1 2
Increments param1 by param2.
DEC{ "Target Register", "Decrement amount"}
Decrement.
Example:
DEC RET2 1
Decrements param1 by param2.
MULT{ "Target Register", "Factor"}
Multiply.
Example:
MULT RET1 2
Multiplies param1 by param2.
DIV{ "Numerator", "Denominator"}
Divide
Example:
DIV RET1 2
Divides param1 by param2.
ANGL { "x1", "y1", "x2","y2"}
Angle
Example:
ANGL POSx POSY BALX BALY
sets RET1 to the angle between two points.
LNPT{ "Distance", "Angle", "x coord","y coord"}
Line Point.
Example:
LNPT BALD BALR POSX POSY
sets RET1 and RET2 to a point at the specified angle and distance from a reference point.
DIST{ "X1", "Y1", "X2","Y2"}
Distance to.
Example:
DIST GOLX GOLY POSX POSY
Calculates the distance between two points.
PUSH{ "stack", "Value"}
Push value to a stack.
Example:
PUSH 1 RET1
Puts value on the stack at the specified stack.
POP{ "stack", "Value"}
Pop.
Example:
POP 1 AUX1
Puts value from the stack to the specified register.
GETO{ "Opponent ID"}
Acquire Opponent data
Example:
GETO 1
Returns position, heading, and speed of opposing bot to RET1, RET2, RET3, and RET4
GETT{ "Team ID"}
Acquire Team data
Example:
GETT 1
Returns position, heading, and speed of bot on your team to RET1, RET2, RET3, and RET4
