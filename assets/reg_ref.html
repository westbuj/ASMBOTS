sBotv2 Registers

POSXread only
Contains the x coordinate of the robot on the soccer field.
POSYread only
Contains the y coordinate of the robot on the soccer field.
BALXread only
Contains the x coordinate of the ball on the soccer field.
BALYread only
Contains the y coordinate of the ball on the soccer field.
BALDread only
Contains the distance to the ball from the robot.
SPDread/write
The speed the robot should move.
DIRread/write
The direction the robot should move. The FCP will read this register and cause the robot to move in the specified direction.

TRGXread/write
Contains the x coordinate of the robot's destination on the soccer field.
If NMOD is set to 1, this registers are ignored. If NMOD is 0, the FCP will set the DIR register for you.
TRGYread/write
Contains the y coordinate of the robot's destination on the soccer field.
If NMOD is set to 1, this registers are ignored. If NMOD is 0, the FCP will set the DIR register for you.

NMODread/write
Instruct the FCP how to navigate the bot
If NMOD is set to 1, this DIR register sets the direction.
If NMOD is 0, the FCP will set the DIR register based on the point in TRGX and TRGY.
GOLXread only
Contains the x coordinate of the goal's center.
GOLYread only
Contains the y coordinate of the goal's center.
GOLDread only
Contains the distance to the goal from the robot.
GOLRread only
Contains the direction of the goal from the robot.
AN1read only
The number of radians in one degree.
AN90read only
The number of radians in 90 degrees.
PIread only
The number of radians in 180 degrees.
USRxread/write
USR0 - USR9
User registers
RETxread/write
RET1 - RET5
Certain FCP functions return values to these registers.
