Basic registers


Some registers can only be read by your programs (read only) and other can be both read and changed (read/write).

POSXread only
Contains the x coordinate of the robot on the soccer field.
POSYread only
Contains the y coordinate of the robot on the soccer field.
BALXread only
Contains the x coordinate of the ball on the soccer field.
BALYread only
Contains the y coordinate of the ball on the soccer field.
BALDread only
Contains the distance to the ball from the robot.
SPDread/write
The speed the robot should move.
DIRread/write
The direction the robot should move. The FCP will read this register and cause the robot to move in the specified direction.
